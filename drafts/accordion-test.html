Accordion Component

This page demonstrates the accordion block with frequently asked questions.

What is Edge Delivery Services?

Edge Delivery Services is a composable set of services that allows for a high degree of flexibility in how you author content on your website. It enables you to deliver exceptional experiences at speed with high performance websites.

How do blocks work?

Blocks are reusable content components that add styling and functionality to your pages. Each block is self-contained with its own JavaScript and CSS files. Authors create blocks using simple table structures in their documents.

What are the performance benefits?

Edge Delivery Services delivers exceptional performance through:

  • Content served from the edge for fast delivery
  • Minimal JavaScript with progressive enhancement
  • Optimized images and assets
  • Three-phase loading for optimal LCP
Can I customize the styling?

Yes! Each block has its own CSS file that you can customize. You can modify colors, spacing, typography, and animations while maintaining the core functionality. All styles are scoped to the block to prevent conflicts.

Is it accessible?

Absolutely. The accordion block is built with accessibility in mind, including proper ARIA attributes, keyboard navigation support, and semantic HTML. It follows WCAG 2.1 AA guidelines.
